
# 🏨 Hotel Management System

A simple **Hotel Management System** built using **Python and Object-Oriented Programming (OOP)** concepts. This project allows users to manage hotel rooms by adding, booking, releasing, and viewing room status.

---

## 📌 Features

* ➕ Add new rooms (Single / Deluxe)
* 📖 Book available rooms
* 🔓 Release booked rooms
* 📊 View all rooms with their current status
* ⚙️ Simple command-line interface

---

## 🛠️ Technologies Used

* Python 3
* OOP Concepts (Classes, Objects, Methods)

---

## 📂 Project Structure

```
hotel-management/
│── main.py        # Main program file
│── README.md      # Project documentation
```

## 🚀 How It Works

The system is based on two main components:

### 1. Room Class

Each room has:

* Room Number
* Room Type (Single / Deluxe)
* Status (Available / Booked)

### 2. Hotel System

Manages:

* Room addition
* Booking logic
* Releasing rooms
* Displaying all room details
